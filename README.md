# DeepRL-Based Autonomous Driving Framework

## Overview

This project focuses on developing an autonomous driving system using Deep Reinforcement Learning (DRL) within the CARLA simulator. The objective is to train an intelligent driving agent capable of learning safe and efficient driving behavior through interaction with a realistic urban environment.

The framework utilizes Proximal Policy Optimization (PPO) and state-based observations to enable the vehicle to learn lane following, road alignment, steering control, and collision avoidance without explicitly programmed driving rules.

🚧 **Project Status: Under Active Development**

This project is currently being developed and improved. New features, reward function enhancements, state representations, and advanced autonomous driving capabilities are continuously being integrated and evaluated.

## Objectives

* Develop an autonomous driving agent using Deep Reinforcement Learning.
* Train the vehicle to perform stable lane-following and lane-centering.
* Learn smooth steering control for straight roads and curved road segments.
* Improve vehicle-road alignment using waypoint-based guidance.
* Implement collision avoidance and safe driving behavior.
* Design and optimize reward functions for robust learning.
* Evaluate PPO-based learning performance in CARLA simulation.
* Enable navigation through urban road environments and intersections.
* Investigate state-based and vision-based observation approaches.
* Build a scalable framework for future integration of V2X communication and advanced ADAS functionalities.
